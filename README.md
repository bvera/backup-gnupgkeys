# backup-gnupgkeys
Simple bash script to export / import GPG Keys

```bash
./backupgpgkeys.sh <export|import> <GnuPG Key>
```

Please note the binary in the script is `gpg2` becase [homebrew](http://brew.sh/), you should probably change it.
